Array Methods

  • indesOf

    • let primary=['red','yellow','blue'];

    format=primary.indexOf("red")//1

  • includes

    • console.log("includes",primary.includes('yellow'))//True;

  • Concat Merge 2 array

    • let secondary=['orange','green','black']; console.log("Concat 2 Array",primary.concat(secondary));

  • reverse Reverse an array

    • console.log("Reverse an array",primary.reverse());

  • Slice Methods copies a portion of an array

    • let colors=['red','yellow','blue','orange','pink','white']; console.log("After slice",colors.slice()); console.log("After slice",colors.slice(2)); console.log("After slice",colors.slice(2,3)); console.log("After slice",colors.slice(-2));

  • Splice removes/replace/add elements in place in original

    • format= splice(start,deleteCount,item0....itemN)

    console.log("After splice",colors.splice(4)); console.log("After splice",colors.splice(0,1)); console.log("After splice",colors.splice(0,1,'black','grey')); console.log("After a splice",colors);

  • Sort Sort the array

    • console.log("Sort array",colors.sort()); let car=["Audi","Maruti","Bmw"]; console.log("Sort array",car.sort());

  • Nested Arrays Array of arrays

    • let numbs=[[2,4],[3,6],[4,8]]; console.log("Nested Arrays ",numbs);
