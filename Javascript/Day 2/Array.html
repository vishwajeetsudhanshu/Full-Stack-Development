Array Methods

  • Push:add to end

    • let car=["Audi","Maruti","Bmw"]; car.push("Toyoto"); console.log("After push",car);

  • pop:Delete from end & returns it

    • car.pop(); console.log("After pop",car); console.log(car);

  • Shift:Delete from Start & returns it

    • car.shift(); console.log("After unshift",car);

  • UnShift:Add to Start

    • car.unshift("Lambo"); console.log("After Shift",car);
